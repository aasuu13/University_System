# 🎓 سامانه مدیریت دانشگاهی

یک نرم‌افزار مدیریت دانشجویان و اساتید با رابط کاربری گرافیکی (GUI) نوشته شده با Python و Tkinter.

---

## ✨ قابلیت‌ها

### برای دانشجو:
- 📚 **انتخاب واحد درسی**: مشاهده لیست دروس، بررسی پیش‌نیازها و انتخاب واحد
- 📊 **مشاهده نمرات و معدل**: نمایش تمام نمرات و محاسبه معدل
- 🏆 **رتبه‌بندی**: مشاهده رتبه خود در بین تمام دانشجویان بر اساس معدل

### برای استاد:
- 📋 **ثبت نمره**: مشاهده لیست دانشجویان هر درس و ثبت نمره
- 📊 **گزارش‌گیری آماری**: مشاهده آمار هر درس (تعداد دانشجو، میانگین، بالا‌ترین و پایین‌ترین نمره)
- 🔄 **ویرایش نمره**: امکان تغییر نمره دانشجویان

### امتیازات ویژه:
- 🔗 **سیستم بررسی پیش‌نیاز**: هنگام انتخاب واحد، وضعیت پیش‌نیاز هر درس نمایش داده می‌شود
- 📈 **گزارش‌گیری پیشرفته**: نمایش درصد ثبت نمرات با رنگ‌بندی

📝 راهنمای استفاده
1. ورود به سیستم

    نام کاربری و رمز عبور را وارد کنید

    بر روی دکمه "ورود" کلیک کنید

2. پنل دانشجو

    📚 انتخاب واحد:

        لیست تمام دروس نمایش داده می‌شود

        وضعیت پیش‌نیاز هر درس مشخص است (✅ قابل انتخاب / ❌ نیاز به پیش‌نیاز)

        فقط دروسی که پیش‌نیاز آنها پاس شده باشد، قابل انتخاب هستند

        با کلیک روی دکمه "انتخاب واحد"، درس به لیست دروس دانشجو اضافه می‌شود

    📊 نمرات و معدل:

        تمام دروس انتخاب شده همراه با نمرات نمایش داده می‌شود

        معدل دانشجو به صورت خودکار محاسبه می‌شود

    🏆 رتبه‌بندی:

        لیست تمام دانشجویان بر اساس معدل مرتب می‌شود

        رتبه خود را در بین سایر دانشجویان مشاهده می‌کنید

3. پنل استاد

    📋 ثبت نمره:

        لیست دروس استاد نمایش داده می‌شود

        برای هر درس، لیست دانشجویان همراه با نمره فعلی نشان داده می‌شود

        با وارد کردن نمره (بین 0 تا 20) و کلیک روی دکمه "ثبت"، نمره ذخیره می‌شود

        پس از ثبت نمره، صفحه به‌روزرسانی می‌شود

    📊 گزارش‌گیری آماری:

        مشاهده آمار هر درس شامل:

            تعداد دانشجویان

            میانگین نمرات

            بالاترین و پایین‌ترین نمره

            درصد ثبت نمرات (با رنگ‌بندی)
